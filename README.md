# Облачное хранилище

## Описание проекта
Разработан REST-сервис. Сервис предоставляет REST-интерфейс для загрузки файлов и вывода списка уже загруженных файлов пользователя.

Все запросы к сервису авторизованы. Авторизация осуществляется через JWT-token.

Сервис разработан с помощью:
* Spring Boot 3.0.7
* Сборщик проекта Maven
* База данных - MySQL
* Для запуска используется Docker

## Возможности
Сервис умеет:
* Выводить список файлов
* Добавлять файл
* Удалять файл
* Авторизовывать пользователя

## Запуск проекта
1. Необходимо скачать проект с репозитория

``https://github.com/Evgenii1323/Cloud.git``

2. Открыть файл с настройками и указать логин и пароль к базе данных

``src/main/resources/application.yaml``

3. Выполнить запуск сервиса, сервис работает на порту 8080

``docker-compose up``
